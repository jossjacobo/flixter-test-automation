# flixter-test-automation-cypress

## Setup

- `npm install`

## Open & Run

- `npm run cypress:open` to open the Cypress GUI or...
- `npm run test` to run the tests

## Starting Point
 
Open the [integration](./cypress/integration/) folder, under the `flixter` folder
